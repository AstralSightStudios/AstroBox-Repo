相信很多小伙伴刚刚更新 AstroBox 2.0，对于很多全新的界面都感到陌生。今天就在这里为你简单介绍一下在全新 UI 中，如何快速开始你的旅程。

## 一些误区

首先要告诉大家一些误区，应用内的所有账号并非必须登录，你可以在未登录的状态下使用应用的大部分功能，包括但不限于连接你的穿戴设备、在穿戴设备上安装各种本地资源以及官方源免费资源。只有当你想要激活一些付费权益、参与评论区讨论、又或是下载米坛资源等时候，登录账号才会派上用场。在账号的注册与登录这件事上，选择权完全交给大家，如果你认为这些登录后才能使用的权益你其实并不需要，那你完全可以不登录、自由地使用这个工具。

包括登录小米账号这件事情，其实也并非强制要求。如果你顾虑心比较重，愿意麻烦一点，完全可以自行提取日志并通过插件市场中的「Authkey提取工具」来获取 Authkey 并连接设备。当然，为了让大家以更低的上手门槛快速开始，我们提供了小米账号登录就能同步设备的功能，保证全过程你的账号密钥只会在你的设备与小米官方账号系统之间传输，我们不会收集这些信息，也对这些信息不感兴趣，让你无需手动提取 Authkey 即可快速同步设备。（此部分也在 Github 完全开源，欢迎前去浏览）

好了，想提前讲的误区就是这些，让我们接下来开始，以最简单的方式连接你的设备并安装资源。

## 连接设备

最简单的连接方式就是直接通过账号同步。我们在设置-账号页底下预留了小米账号与 vivo 账号登录同步设备的功能，你可以直接前去登录并同步设备。

![设置中的账号页预留的登录按钮](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/1.png)

当然，在设备列表里也预留了一个同步按钮，在这里也可以完成这个操作。设备列表的位置对于手机来说就在你的正下方，此时应该大大的写着「设备未连接」，点击进去就有切换设备按钮；对于电脑来说则是在左边偏上部分，点击就是设备列表。

![各端设备列表位置](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/2.png)
![设备卡片中的「通过账号同步」按钮](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/3.png)

在你完成设备同步之后，你就可以开始连接设备了。首先进入穿戴设备的设置页，找到「我的设备」等相关设置页面，通常这个页面是控制设备重启、关机的。随后请你找到并点击「连接新手机」选项。

按步骤完成之后，穿戴设备将进入连接新手机状态，此时你就可以点击 AstroBox 中设备列表里的对应设备，开始连接了。如果你的手机/电脑出现配对提示，请点击绑定。如果是安卓手机，提示也可能以上方通知的形式展示，点击通知中的配对与连接即可。

接下来绑定就完成了～你可以开始体验各种穿戴设备三方功能了～

## 安装本地资源

连接穿戴设备后，你就可以直接在应用内安装本地下载好的资源。在这里我们介绍两种方式：在设备页安装与在队列中安装。

### 在设备页安装

首先通过菜单打开设备页，电脑端在左侧导航栏中点击「设备信息」按钮，手机则是点击下方的胶囊形设备卡片。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/11.png)
![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/12.png)

然后你需要在连接了设备的情况下通过点击卡片，进入想要安装的资源类型的子页面。例如如果你想要安装表盘，你可以点击设备页面中的表盘卡片。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/13.png)

进入子页面后，点击右上角加号，你可以选择资源文件并安装。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/14.png)

如果你关闭了本地资源自动安装，那么你还需要进入队列页面。队列页面一般在电脑端的左下角、手机端的胶囊右侧。请注意，手机端的队列按钮仅在连接了设备后才会显示，如果你没找到它，请确保当前处于设备连接成功的状态。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/15.png)

此时点击队列上方的开始键就可以开始资源的推送。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/16.png)

### 在队列中安装

首先你需要进入队列页面。队列页面一般在电脑端的左下角、手机端的胶囊右侧。请注意，手机端的队列按钮仅在连接了设备后才会显示，如果你没找到它，请确保当前处于设备连接成功的状态。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/15.png)

接着在连接了设备的情况下点击队列右上角的加号，选择你要安装的资源类型，然后选择资源文件并安装。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/17.png)

如果你关闭了本地资源自动安装，你还需要点击队列上方的开始键手动开始资源的推送。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/16.png)

## 安装官方源资源

AstroBox 官方源中包括了很多出彩的资源。你可以从探索页直接查看我们推荐的优质作品，也可以点击进入全部资源，查看整个 AstroBox 社区中所有人的作品。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/4.png)
![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/52.png)

在完成设备连接的前提下，进入你心仪的一个资源的详情页，点击下载，即可将适配你当前设备的资源安装到你的设备上。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/6.png)

如果遇到资源未适配你当前使用的设备，我们也会给与提示。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/7.png)

在提示资源未适配的情况下，我们十分不建议你强制安装，因为这可能会带来一些意想不到的问题，重则影响穿戴设备正常使用。建议您提前询问开发者，如果明确了某一个版本可以安装并能达到符合预期的运行效果，那么你可以在右侧选择版本，下载与安装会立刻开始。

![](https://raw.githubusercontent.com/AstralSightStudios/AstroBox-Repo/refs/heads/main/blogs/docs/8.png)
